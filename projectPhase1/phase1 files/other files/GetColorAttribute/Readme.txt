

ChangeJson.sh is used to add color attribute to the existing json files.
Since json is immutable new json object is created and a result file is generated.

this  program takes 2 command line arguments. 1st argument is the json file generated by the crawler program. 2nd argument is the text file which contains imagefilename with colors. 


Sample inputs to be given are also added in the folder.

****************************************************************************

usage: startJsonColor.sh <input Json file> < image color textfile>
